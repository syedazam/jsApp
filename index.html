Learning OOPS ?

In OOPs programs are considered as a "collection of objects". Each object is nothing but an instance of a class.
* For Understanding OOPs, you should be familiar with some of JavaScript concepts such as

variables : var - It holds the value, it should be declared with “var” keyword, you can declare multiple variables at once, until you assign a value to variable it is “undefined”. variable can be a string ("alphebets or numbers within single / double quotes") / number (without quotes) / Boolean (true/false) / null("") or undefined.
